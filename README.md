# 📝 Python To-Do App using Docker

## 📌 Objective
This project is a simple Python-based To-Do application built using Flask.  
The application is containerized using Docker and managed using Docker Compose.  
It allows users to add, edit, delete, and mark tasks as completed.

---

## 🚀 Features
- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Mark tasks as completed (cross-off)  
- Simple and clean UI (Bootstrap)

---

## 🛠️ Technologies Used
- Python
- HTML  
- Flask  
- Docker  
- Docker Compose

## ⚙️ How to Run the Application

 1 Clone the repository
```bash
git clone https://github.com/abhishekpsuresh/python-todo-docker.git
cd python-todo-docker
```
### 2️ Run using Docker Compose
```bash
docker compose up --build
```
### 3 Open in browser
```bash
http://localhost:5000
```
